# Kape Kunana Design System

## Purpose

This document is the source of truth for all design decisions.

Every page, component, animation, and layout must follow these rules.

---

# Design Philosophy

Kape Kunana should feel:

* Premium
* Warm
* Modern
* Inviting
* Minimal
* Coffee-first
* Community-driven

Design should feel similar in quality to premium specialty coffee brands.

Avoid generic templates.

---

---

# Typography

## Headings

Modern Sans Serif

Characteristics:

* Bold
* Clean
* Premium
* Highly readable

Use strong visual hierarchy.

Never use decorative fonts.

## Body

Clean sans-serif.

Readable on mobile.

Maximum readability is preferred over creativity.

---

# Layout Rules

Use generous whitespace.

Avoid crowded sections.

Every section must have clear separation.

Preferred container width:

1200px - 1400px

Preferred spacing:

Section Padding:
120px desktop
80px tablet
60px mobile

---

# Grid System

Desktop:
12 columns

Tablet:
8 columns

Mobile:
4 columns

Always design mobile-first.

---

# Buttons

Primary CTA:

* Filled
* High contrast
* Rounded corners
* Clear action text

Examples:

* Order Now
* View Menu
* Visit Us

Secondary CTA:

* Outline style
* Lower visual weight

Never use more than 2 CTAs per section.

---

# Cards

Cards should:

* Have breathing room
* Use subtle shadows
* Have soft radius
* Feel premium

Avoid heavy borders.

---

# Imagery

Use:

* Coffee preparation
* Espresso shots
* Latte art
* Coffee beans
* Human moments
* Community moments

Avoid:

* Generic stock photos
* Overly staged imagery

---

# Motion Design

Animation should feel smooth and premium.

Use GSAP.

Preferred effects:

* Fade Up
* Scale In
* Stagger Reveal
* Scroll Trigger
* Text Reveal

Avoid:

* Bounce
* Flashing effects
* Excessive motion

Animation duration:

0.6s - 1.2s

Use easing.

---

# Hero Section Standards

Must include:

* Full-screen layout
* Video or premium imagery
* Strong headline
* Short supporting text
* Primary CTA
* Secondary CTA
* Scroll indicator

Hero should immediately communicate:

"Why Kape Kunana is worth visiting."

---

# Responsive Rules

Mobile experience is a priority.

No horizontal scrolling.

Text must remain readable.

Buttons must be thumb-friendly.

Minimum touch target:
44px

---

# Accessibility

Maintain strong color contrast.

Use semantic HTML.

Provide alt text for images.

Never rely on color alone.

---

# Quality Checklist

Before completing any design:

✓ Premium appearance
✓ Mobile responsive
✓ Accessible
✓ Consistent spacing
✓ Strong hierarchy
✓ Brand aligned
✓ Performance friendly
✓ Modern 2026 standards
